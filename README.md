## Learning Objectives

By the end of the course, you will have knowledge of:

- Basic HTML tags  
- HTML document structure  
- How HTML works  
- How CSS works  
- Chrome DevTools  
- HTML input fields  
- Anchor tags  
- Deployment with Netlify  
- Basic CSS syntax  
- Centering with CSS  
- Basic CSS Flexbox  
- Padding in CSS  
- Margins in CSS  
- Borders in CSS  
- Getting started with GitHub
